## Rating Scale

| Score | Meaning                                                     |
| ----: | ----------------------------------------------------------- |
|     0 | No exposure                                                 |
|     1 | Recognizes terms / copied examples only                     |
|     2 | Can run or modify with guidance                             |
|     3 | Can complete small tasks with help and debugging support    |
|     4 | Can complete simple tasks independently in familiar context |
|     5 | Reliable beginner / early junior fundamentals               |
|     6 | Can debug and adapt moderate tasks                          |
|     7 | Junior job-task ready in this area                          |
|     8 | Strong independent practical skill                          |
|     9 | Advanced professional skill                                 |
|    10 | Expert / specialist level                                   |
